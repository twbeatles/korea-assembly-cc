# 🏛️ 국회 의사중계 자막 추출기 v11.0

국회 의사중계 웹사이트에서 실시간 AI 자막을 자동으로 추출하고 저장하는 프로그램입니다.

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![PyQt6](https://img.shields.io/badge/PyQt6-6.0+-green.svg)
![Selenium](https://img.shields.io/badge/Selenium-4.0+-orange.svg)

---

## ✨ 주요 기능

- **실시간 자막 추출** - 국회 의사중계 웹사이트의 AI 자막을 실시간으로 캡처
- **다양한 저장 형식** - TXT, SRT, VTT, DOCX 지원
- **키워드 하이라이트** - 특정 단어 강조 표시
- **키워드 알림** - 특정 단어 감지 시 토스트 알림
- **헤드리스 모드** - 브라우저 창 숨기고 백그라운드 실행
- **자동 백업** - 5분마다 자동 백업 (최대 10개 유지)
- **세션 저장/불러오기** - 작업 중단 후 재개 가능
- **다크/라이트 테마** - 사용자 선호에 맞는 테마 선택

---

## 📦 설치

### 필수 라이브러리

```bash
pip install PyQt6 selenium
```

### 선택 라이브러리 (DOCX 저장용)

```bash
pip install python-docx
```

### Chrome WebDriver

Chrome 브라우저와 버전에 맞는 ChromeDriver가 필요합니다.
- [ChromeDriver 다운로드](https://chromedriver.chromium.org/downloads)

> **참고**: Selenium 4.0 이상에서는 자동으로 ChromeDriver를 관리합니다.

## 🚀 실행

```bash
python "국회의사중계 자막.py"
```

---

## 📋 사용법

1. **URL 입력** - 국회 의사중계 페이지 URL 입력
2. **선택자 확인** - 기본값 사용 또는 필요시 수정
3. **옵션 설정**
   - ☑️ 자동 스크롤: 새 자막 자동 따라가기
   - ☑️ 실시간 저장: 자막 실시간 파일 저장
   - ☑️ 헤드리스 모드: 브라우저 창 숨기기
4. **시작** 버튼 클릭 (또는 `F5`)
5. 자막 추출 완료 후 **파일 저장**

---

## ⌨️ 단축키

| 단축키 | 기능 |
|--------|------|
| `F5` | 시작 |
| `Escape` | 중지 |
| `Ctrl+F` | 검색 |
| `F3` / `Shift+F3` | 다음/이전 검색 |
| `Ctrl+T` | 테마 전환 |
| `Ctrl+S` | TXT 저장 |
| `Ctrl++` / `Ctrl+-` | 글자 크기 조절 |
