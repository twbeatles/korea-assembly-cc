# 🏛️ 국회 의사중계 자막 추출기 v14.0

국회 의사중계 웹사이트에서 **실시간 AI 자막**을 자동으로 추출하고 저장하는 PyQt6 기반 프로그램입니다.

## ✨ v14.0 새 기능

### 🖥️ HiDPI 지원
- **고해상도 디스플레이** 완벽 지원
- 자동 DPI 스케일링

### 🔧 디버깅 및 안정성 강화
- **자막 셀렉터 자동 감지** - 웹사이트 구조 변경에 대응
- **개선된 자막 활성화** - 5가지 방법으로 AI 자막 자동 활성화
- **토스트 알림 스택 처리** - 알림 겹침 방지
- **종료 시 저장 취소** 버그 수정

### 🔍 UI/UX 개선
- **검색바 닫기(✕) 버튼** 복원
- 타입 힌트 추가로 코드 품질 향상
- 중복 코드 리팩토링

## ✨ 주요 기능

### 🎯 실시간 자막 추출
- 국회 의사중계 웹사이트의 AI 자막을 실시간으로 캡처
- **2초 타임아웃**으로 자막 자동 확정
- 중복 자막 자동 제거 및 병합
- **자동 셀렉터 감지** - 웹사이트 변경에 자동 대응

### 💾 다양한 저장 형식
| 형식 | 설명 | 단축키/라이브러리 |
|------|------|-----------------|
| **TXT** | 일반 텍스트 | Ctrl+S |
| **SRT** | 자막 파일 형식 | - |
| **VTT** | WebVTT 자막 형식 | - |
| **DOCX** | Word 문서 | python-docx 필요 |
| **HWP** | 한글 문서 | Hancom Office + pywin32 |
| **RTF** | 서식 있는 텍스트 | 기본 (HWP 호환) |

### 🔍 검색 및 하이라이트
- **실시간 검색** - 자막 내 텍스트 검색 (Ctrl+F)
- **키워드 하이라이트** - 특정 단어 강조 표시
- **알림 키워드** - 특정 키워드 감지 시 토스트 알림

### 🏛️ 상임위원회 프리셋
- **📋 상임위 버튼** - 16개 상임위원회 URL 빠른 선택
- **사용자 정의 프리셋** 추가/관리 기능

### ⚙️ 편의 기능
- **헤드리스 모드** - 브라우저 창 숨기고 백그라운드 실행
- **실시간 저장** - 자막을 파일에 자동 저장
- **세션 저장/불러오기** - 작업 내용 보존
- **URL 히스토리** - 최근 사용한 URL 자동 저장 (태그 지원)
- **자동 백업** - 5분마다 자동 백업
- **창 위치 기억** - 프로그램 종료 시 창 위치/크기 저장

### 🎨 UI 기능
- **다크/라이트 테마** 전환 (Ctrl+T)
- **타임스탬프 표시** 옵션
- **자동 스크롤** 옵션
- **통계 패널** - 글자 수, 단어 수, 분당 글자 수
- **토스트 알림** - 비차단 알림 메시지 (스택 처리)

## 📋 사용 방법

### 기본 사용
1. **URL 입력** - 국회 의사중계 페이지 URL 입력 (또는 상임위 프리셋 선택)
2. **선택자 확인** - 기본값 `#viewSubtit .incont` 사용 (자동 감지 지원)
3. **옵션 설정**
   - 자동 스크롤: 새 자막 자동 따라가기
   - 실시간 저장: 자막 실시간 파일 저장
   - 헤드리스 모드: 브라우저 창 숨김
4. **시작 버튼** 클릭 또는 F5
5. 자막 추출 완료 후 원하는 형식으로 **파일 저장**

### HWP 저장 방법
1. 자막 수집 완료 후 메뉴 → **파일** → **HWP 저장 (한글)**
2. Hancom Office가 설치되어 있으면 HWP 파일 직접 생성
3. 설치되어 있지 않으면 자동으로 RTF 형식으로 저장

> 💡 **Tip**: RTF 파일은 한글(HWP) 프로그램에서 바로 열 수 있습니다.
