# 🏛️ 국회 의사중계 자막 추출기

국회 의사중계 웹사이트에서 **실시간 AI 자막**을 자동으로 추출하고 저장하는 PyQt6 기반 프로그램입니다.

## ✨ 주요 기능

### 🎯 실시간 자막 추출
- 국회 의사중계 웹사이트의 AI 자막을 실시간으로 캡처
- **2초 타임아웃**으로 자막 자동 확정
- 중복 자막 자동 제거 및 병합

### 💾 다양한 저장 형식
- **TXT** - 일반 텍스트 (Ctrl+S)
- **SRT** - 자막 파일 형식
- **VTT** - WebVTT 자막 형식
- **DOCX** - Word 문서 (python-docx 필요)

### 🔍 검색 및 하이라이트
- **실시간 검색** - 자막 내 텍스트 검색 (Ctrl+F)
- **키워드 하이라이트** - 특정 단어 강조 표시

### ⚙️ 편의 기능
- **헤드리스 모드** - 브라우저 창 숨기고 백그라운드 실행
- **실시간 저장** - 자막을 파일에 자동 저장
- **세션 저장/불러오기** - 작업 내용 보존
- **URL 히스토리** - 최근 사용한 URL 자동 저장

### 🎨 UI 기능
- **다크/라이트 테마** 전환 (Ctrl+T)
- **타임스탬프 표시** 옵션
- **자동 스크롤** 옵션
- **통계 패널** - 글자 수, 단어 수, 분당 글자 수

## 📦 설치

### 필수 라이브러리
```bash
pip install PyQt6 selenium
```

### 선택 라이브러리 (DOCX 저장용)
```bash
pip install python-docx
```

### Chrome WebDriver
- Selenium 4.6+ 버전은 자동으로 ChromeDriver를 관리합니다
- 또는 [ChromeDriver](https://chromedriver.chromium.org/) 수동 설치

## 🚀 실행

```bash
python "251211 국회의사중계 자막.py"
```

## ⌨️ 단축키

| 기능 | 단축키 |
|------|--------|
| 시작 | F5 |
| 중지 | Escape |
| 검색 | Ctrl+F |
| 다음 검색 | F3 |
| 이전 검색 | Shift+F3 |
| 테마 전환 | Ctrl+T |
| TXT 저장 | Ctrl+S |
| 세션 저장 | Ctrl+Shift+S |
| 세션 불러오기 | Ctrl+O |
| 종료 | Ctrl+Q |

## 📋 사용 방법

1. **URL 입력** - 국회 의사중계 페이지 URL 입력
2. **선택자 확인** - 기본값 `#viewSubtit .incont` 사용
3. **옵션 설정**
   - 자동 스크롤: 새 자막 자동 따라가기
   - 실시간 저장: 자막 실시간 파일 저장
   - 헤드리스 모드: 브라우저 창 숨김
4. **시작 버튼** 클릭 또는 F5
5. 자막 추출 완료 후 **파일 저장** (TXT, SRT, VTT, DOCX)
