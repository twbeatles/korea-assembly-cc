Python 3.10 이상이 필요합니다.

### 2. 필수 라이브러리 설치
```bash
pip install PyQt6 selenium
```

### 3. 선택 라이브러리 (추가 기능)

## � 사용 방법

### 기본 사용법

#### 1단계: 프로그램 실행
```bash
python "251226 국회의사중계 자막.py"
```

#### 2단계: 위원회 선택
두 가지 방법 중 선택:

**방법 A: 프리셋 사용 (권장)**
1. `📋 상임위` 버튼 클릭
2. 원하는 위원회 선택 (예: 본회의, 법제사법위원회)
3. URL이 자동으로 입력됨

**방법 B: 직접 URL 입력**
1. URL 입력창에 국회 의사중계 URL 입력
2. 예: `https://assembly.webcast.go.kr/main/player.asp?xcode=10`

#### 3단계: 옵션 설정
- ✅ **자동 스크롤**: 새 자막이 추가될 때 자동으로 아래로 스크롤
- ✅ **실시간 저장**: 자막을 파일에 실시간으로 저장 (realtime_output 폴더)
- ✅ **헤드리스 모드**: 브라우저 창을 숨기고 백그라운드에서 실행

#### 4단계: 추출 시작
- `▶ 시작` 버튼 클릭 또는 **F5** 키
- 상태바에서 진행 상황 확인:
  - `🔍 현재 생중계 감지 중...` - xcgcd 자동 탐지 진행
  - `✅ 생중계 감지 성공!` - 오늘 생중계 발견
  - `자막 모니터링 중` - 정상 동작 중

#### 5단계: 자막 확인
- 왼쪽 패널에 자막이 실시간으로 표시됨
- 오른쪽 통계 패널에서 글자수, 단어수, 실행 시간 확인
- `⏳` 표시는 아직 확정되지 않은 진행 중 자막

#### 6단계: 저장
추출 완료 후 다양한 형식으로 저장:
- **파일 → TXT 저장** (Ctrl+S) - 일반 텍스트
- **파일 → SRT 저장** - 자막 파일 형식
- **파일 → DOCX 저장** - Word 문서
- **파일 → 세션 저장** (Ctrl+Shift+S) - 나중에 다시 불러오기 가능

### 고급 사용법

#### 키워드 하이라이트
1. **편집 → 하이라이트 키워드 설정**
2. 키워드 입력 (쉼표로 구분): `법안, 통과, 의결`
3. 해당 키워드가 노란색으로 강조됨

#### 알림 키워드
1. **편집 → 알림 키워드 설정**
2. 키워드 입력: `긴급, 중요`
3. 해당 키워드가 포함된 자막 감지 시 토스트 알림 표시

#### 자막 편집/삭제
- **Ctrl+E**: 자막 편집 다이얼로그 열기
- **Delete**: 자막 삭제 (다중 선택 가능)

#### 세션 관리
- **Ctrl+Shift+S**: 현재 작업을 JSON으로 저장
- **Ctrl+O**: 이전 세션 불러오기

#### 트레이 모드
1. **보기 → 트레이로 최소화** 활성화
2. 창을 닫으면 트레이로 최소화됨
3. 트레이 아이콘 더블클릭으로 복원

## 📚 상세 기능 설명

### 생중계 자동 감지 (v16.1)
xcode만 포함된 URL로 접속하면 프로그램이 자동으로 현재 진행 중인 생중계를 찾습니다.

**작동 원리:**
1. 페이지의 JavaScript 변수에서 xcgcd 확인
2. URL 리다이렉트 감지
3. 페이지 내 링크/iframe/input 요소 파싱
4. 메인 페이지에서 해당 위원회의 생중계 링크 조회

**상태 표시:**
- `🔍 현재 생중계 감지 중...` - 감지 진행 중
- `✅ 생중계 감지 성공!` - xcgcd 발견
- `⚠️ 생중계 정보를 찾을 수 없음` - 감지 실패 (기본 URL 사용)

### 상임위원회 프리셋
20개 상임위원회의 URL이 내장되어 있습니다:

- 본회의, 운영위원회, 법제사법위원회, 정무위원회
- 기획재정위원회, 교육위원회, 과학기술정보방송통신위원회
- 외교통일위원회, 국방위원회, 행정안전위원회
- 문화체육관광위원회, 농림축산식품해양수산위원회
- 산업통상자원중소벤처기업위원회, 보건복지위원회
- 환경노동위원회, 국토교통위원회, 여성가족위원회
- 예산결산특별위원회, 특별위원회, 청문회/공청회

**약칭 지원:** 법사위, 기재위, 과방위, 행안위, 문체위 등

### 통계 기능
실시간으로 다음 정보를 표시합니다:
- ⏱️ 실행 시간
- 📝 글자 수
- 📖 단어 수
- 💬 문장 수
- ⚡ 분당 글자 수 (CPM)

**통계 내보내기** (파일 → 통계 내보내기)로 상세 보고서 생성 가능

## 🔧 문제 해결

### Chrome 시작 오류
- Chrome 브라우저가 최신 버전인지 확인
- 프로그램이 자동으로 최대 3회 재시도합니다
- Selenium 4.x가 WebDriver를 자동 관리합니다

### 자막이 표시되지 않음
1. 국회 의사중계 페이지에서 자막이 활성화되어 있는지 확인
2. 다른 CSS 선택자 시도: `#viewSubtit`, `.subtitle_area`
3. 생중계가 진행 중인지 확인

### xcgcd 감지 실패
- 현재 진행 중인 생중계가 없을 수 있습니다
- 프로그램은 기본 URL(마지막 방송)을 사용합니다
- 직접 생중계 URL을 입력하면 됩니다

### HWP 저장 오류
- **한글 오피스**가 설치되어 있어야 합니다
- COM 오류 시 자동으로 RTF 형식으로 저장됩니다
- RTF 파일은 한글에서 열 수 있습니다

### 헤드리스 모드에서 자막 미감지
- 일부 JavaScript 요소가 지연 로딩될 수 있습니다
- 헤드리스 모드를 해제하고 다시 시도해보세요

## 🛠️ 빌드 방법

### PyInstaller로 EXE 빌드
```bash
# PyInstaller 설치
pip install pyinstaller

# 빌드 실행
pyinstaller subtitle_extractor.spec

# 결과물
dist/subtitle_extractor.exe  # 단일 실행 파일 (~80MB)
```

- `console=True` - 콘솔 창 표시 (디버깅용)
